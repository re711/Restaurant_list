# Restaurant_list
使用Node.js + Express 打造的餐廳美食網站，可依照餐廳名稱與類別進行搜尋。

# 安裝流程
```
 1. 終端機輸入 git clone https://github.com/re711/Restaurant_list.git
 ```
 ```
 2. 安裝套件 npm install
 ```
 ```
 3. 執行 npm run dev
 ```
`
# 功能
1. 使用者可以點擊任一餐廳，查看更多餐廳資訊，如地址、電話與簡介
2. 使用者可以依照中文名稱、英文名稱與餐廳類別進行搜尋

# 工具
* Node.js
* Express
* Express-handlebars
* Bootstrap
